# 选型建议速查


| 你的需求           | 推荐 LLM                | 推荐 Embedding             | 理由                        |
| -------------- | --------------------- | ------------------------ | ------------------------- |
| 追求极致智能/复杂Agent | GPT-5.4 / Claude 4.6  | text-embedding-3-large   | 逻辑推理与任务规划能力目前最强。          |
| 处理超长文档/书籍      | Kimi 2 / Gemini 3.1   | text-embedding-004       | 上下文窗口最大，不易丢失细节。           |
| 中文场景/高性价比      | DeepSeek-V3.2 / Qwen3 | BGE-M3 / Qwen3-Embedding | 中文理解地道，价格仅为美系模型的1/10甚至更低。 |
| 数据隐私/私有化部署     | Llama 4 / Qwen3 (开源版) | BGE-M3 / Llama-Embedding | 可下载权重本地运行，数据不出域。          |
| 代码开发辅助         | Claude 4.6 / GPT-5.4  | *无特殊要求*                  | 代码生成与Debug能力经过大量验证。       |


2026年的趋势是**“模型专用化”**。大多数企业不再试图用一个模型解决所有问题，而是采用 **“高性能嵌入模型（检索） + 垂直领域大模型（生成）”** 的组合拳模式，以实现成本与效果的最优平衡。
